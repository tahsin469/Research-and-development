# **.NET Backend Roadmap **

---

## **1️⃣ Developer Foundations (ডেভেলপার বেসিকস)**

**উদ্দেশ্য:** প্রোগ্রামিং, ওয়েব, ডেটা স্ট্রাকচার ও সমস্যা সমাধানের ভিত্তি তৈরি।

### **Topics Covered:**

* Internet & Networking: TCP/IP, DNS, HTTP/HTTPS, SSL/TLS
* Web Basics: Client-Server Architecture, REST API, Stateless, GraphQL, gRPC
* HTTP Status Codes & Error Handling (200, 201, 400, 401, 403, 404, 500)
* OOP Principles: Class, Object, Encapsulation, Inheritance, Polymorphism, Abstraction
* Problem Solving & Basic DSA: Array, Stack, Queue, Linked List, Trees, Graphs, Searching & Sorting
* Git & GitHub: Branching, Merging, Rebase, Pull Requests
* ChatGPT / AI Tools for Code Review, Refactoring & Debugging

### **Interview Questions:**

1. HTTP vs HTTPS – Security difference explain করুন।
2. Stateless API কি এবং কেন গুরুত্বপূর্ণ?
3. GET vs POST পার্থক্য উদাহরণসহ।
4. Stack vs Heap Memory explain করুন।
5. Git merge vs rebase পার্থক্য।
6. REST API design best practices কী?

## **4️⃣ Database & ORM (EF Core + Dapper)** ✅

**উদ্দেশ্য:** ডেটাবেস দক্ষতা, ORM ব্যবহারে দক্ষতা এবং Query Optimization।

### **Topics Covered:**

* **EF Core:** Code First, Database First, Migrations, DbContext Lifetime, LINQ, Deferred Execution
* Loading Strategies: Lazy, Eager, Explicit Loading
* Interceptors, Global Query Filters, Schemas
* Transaction & Concurrency Handling
* Performance Optimization Techniques
* **Dapper:** Raw SQL Queries, Multi-Mapping, Stored Procedures
* **NoSQL:** MongoDB, DynamoDB
* **Caching:** Redis (In-Memory, Sliding/Absolute Expiration, Cache Invalidation)
* SQL Concepts: CRUD, Joins, Index, ACID Properties, Deadlocks, Query Optimization, Stored Procedures, Triggers

### **Interview Questions:**

1. EF Core vs Dapper – Trade-offs কী?
2. Lazy Loading vs Eager Loading – কখন ব্যবহার করবেন?
3. LINQ Deferred Execution explain করুন।
4. Transaction & Concurrency Handling practical উদাহরণ।
5. Redis Cache Invalidation Techniques explain করুন।
6. DbContext Lifetime vs Thread Safety explain করুন।
